API 接口文档

所有接口均支持 GET 请求，返回统一的 JSON 格式数据。支持跨域调用 (CORS)，可直接在前端项目中使用。

网易云音乐

GET
ENDPOINT /api/netease

请求示例：

/api/netease?input=十年&filter=name&page=1

参数说明：

  • input: 搜索关键词或歌曲ID (必填)
  • filter: name(名称) | id(ID) (默认 name)
  • page: 页码 (默认 1)

QQ音乐

GET
ENDPOINT /api/qq

请求示例：

/api/qq?input=告白气球&filter=name

酷狗音乐

GET
ENDPOINT /api/kugou

请求示例：

/api/kugou?input=周杰伦&filter=name

酷我音乐

GET
ENDPOINT /api/kuwo

请求示例：

/api/kuwo?input=陈奕迅&filter=name
